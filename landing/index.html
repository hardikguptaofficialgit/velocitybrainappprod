Simple, Powerful API

Integrate VelocityBrain in minutes with our RESTful API. Store memory, execute skills, and retrieve context with simple HTTP requests.

  • RESTful API with JSON responses
  • SDKs for Python, JavaScript, and more
  • WebSocket support for real-time updates
// Store memory for your agent
await fetch('/v1/memory/store', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    agentId: 'agent-123',
    content: 'User prefers dark mode',
    metadata: { type: 'preference' }
  })
});
